# local-server-tools

ローカルサーバで動作するちょっとしたツール類

## Environment

- OS:
    - Windows 10
    - Ubuntu 18.04
- Node.js: 12.14.1
    - Yarn package manager: 1.21.1
        - Install command: `npm install -g yarn`

### Setup
```bash
# install: node_modules
$ yarn install

# start: zero server
$ yarn zero
```

Zero Server が http://localhost:3000 で稼働する

### 備忘録
```bash
# initialize project
$ yarn init -y

# install: zero server
$ yarn add zero
```

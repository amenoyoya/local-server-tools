# Zero Server を使ったちょっと便利なWEBアプリ

## Zero Server とは

- 公式ページ: https://zeroserver.io/
    - GitHub: https://github.com/remoteinterview/zero
- WEB開発をシンプルにするための Express ベースのフレームワーク
- React, Vue といった JavaScript フレームワークを、面倒な導入作業や設定なしに使うことが可能
